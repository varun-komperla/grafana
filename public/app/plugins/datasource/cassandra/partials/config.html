User Permission

The user should only be granted SELECT permissions on the specified keyspace & tables you want to query. Grafana does not validate that queries are safe so queries can contain any CQL statement. For example, statements like USE otherkeyspace; and DROP TABLE table; would be executed. To protect against this we Highly recommend you create a specific Cassandra user with restricted permissions. Checkout the Cassandra Data Source Docs for more information.
